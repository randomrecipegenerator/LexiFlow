🎬 LexiFlow + CRM Sync

2-Minute Silent Demo Video — Storyboard

This document describes each scene for the silent demo video showing a lead being qualified and synced to CRM.

0:00–0:10 Scene 1: Lead Arrives via Web Widget

A prospect visits the law firm's website. The LexiFlow intake widget appears in the bottom-right corner. The user types "I was in a car accident" and submits.
Visual: Browser window → LexiFlow widget pops up → User types message → "AI is reviewing..." indicator
Overlay: "Lead Captured — AI Qualifying"

0:10–0:25 Scene 2: AI Qualification in Action

Cut to the LexiFlow dashboard showing the lead being analyzed in real-time. The AI Merit Score appears and climbs from 45 → 82/100 as the system processes liability, damages, and causation signals.
Visual: Dashboard screen → Lead card appears → Score animates upward → Signal bars fill (Liability: 92%, Damages: 65%, Causation: 88%)
Overlay: "Reasoning AI — 87/100 — High Value Lead"

0:25–0:40 Scene 3: Auto-Create CRM Contact

The system automatically pushes the qualified lead to Clio Grow. Watch as a new contact and matter are created in real-time with all lead data populated.
Visual: Split screen — Left: LexiFlow showing "Syncing to Clio Grow..." Right: Clio interface showing new contact appearing → matter created
Overlay: "Contact Created ✓ — Matter Created ✓"

0:40–0:55 Scene 4: Filevine Sync

The same lead is now pushed to Filevine. A case file is created with the AI qualification report attached. Text message shows "Lead synced to Filevine ✓"
Visual: Switch to Filevine interface → Case appears → AI report attached as document → Status changes to "Qualified"
Overlay: "Multi-CRM Sync — Clio + Filevine"

0:55–1:10 Scene 5: Activity Log

Show the CRM sync activity feed with timestamps. Each step is logged: capture → qualification → Clio sync → Filevine sync. Green checkmarks appear next to each completed action.
Visual: Scrollable activity log → Items appear with timestamps → All green ✓
Overlay: "Full Audit Trail — Every Action Logged"

1:10–1:25 Scene 6: Integration Settings

Show the CRM Integration Configuration page. Three CRM cards (Clio, Filevine, LeadDock) with connection statuses. Toggle and Settings buttons visible.
Visual: Integration settings page → Connected status indicators → Toggle ON/OFF → Settings gear icon
Overlay: "Enterprise-Grade Integrations"

1:25–1:45 Scene 7: Medical Chronology Sync

A medical chronology generated by LexiFlow is synced directly to the CRM case file. Show the AI chronology on the left and Clio/Filevine on the right with "Attach to Matter" action.
Visual: Left: AI chronology timeline → Right: Clio document attachments → "Attached ✓" confirmation
Overlay: "AI Chronology → CRM in One Click"

1:45–2:00 Scene 8: Summary & CTA

Show the final summary: 3 CRM integrations active, 2 leads synced today, 99.8% uptime. End screen with "Get Started — $29/mo" and mailto CTA.
Visual: Summary dashboard → Stats counters → LexiFlow logo → "info@lexiflow.co" CTA
Overlay: "All 6 Modules — $29/mo Flat — HIPAA Compliant"

📹 Production Notes

Use screen recording software (OBS, ScreenFlow, or Loom) at 1920×1080.
Background music: Low-fi or ambient instrumental (royalty-free).
