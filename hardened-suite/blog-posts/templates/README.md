# Blog Post Template Usage

Copy `post-template.html` to create new blog posts. Replace the following placeholders:

| Placeholder | Description |
|---|---|
| `{title}` | Blog post title (SEO-optimized) |
| `{excerpt}` | Meta description excerpt |
| `{category}` | Category name (AI Intake, Medical Chronologies, etc.) |
| `{date}` | Publication date (e.g., Jun 1, 2025) |
| `{read_time}` | Read time in minutes |
| `{emoji}` | Featured image emoji/icon |
| `{content}` | Full HTML content of the post |
| `{cta_subject}` | URL-encoded CTA subject line |
